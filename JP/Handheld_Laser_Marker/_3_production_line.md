## 2.1. 静止スプレーコード設定

1. パイプライン設定
   パイプライン方向で「Stop（停止）」を選択します（図3-1）。
   図3-1

2. マーキングモード設定
   マーキングモードは「Jog mode」または「Sensor/Foot mode」を選択できます。

### 2.1.1. Jog mode

マーキングモードを「Jog mode」に変更します（図3-2）。

Foot Switch：
モードがJog modeのとき、フットスイッチをトリガとして選択できます。フットフィルタを設定でき、Startをクリックしてからフットスイッチ信号が入るとコード印字を行います。

Cont. mode：
チェックされている場合は、Startをクリックすると設定した時間間隔で連続してコード印字を行います。チェックされていない場合は、Startをクリックするごとに1回だけコード印字を行います。

Red light mode：
マーキング完了後、自動的に赤色ガイドライトを点灯させます。

図3-2

### 2.1.2. Sensor / Foot mode

マーキングモードを「Sensor / Foot mode」に変更します（図3-3）。
このモードではセンサーを有効にする必要があり、オンにしないとトリガ信号を受信できません。このモードではフットスイッチは有効になりません。

Open sensor：
必ずチェックを入れてください。チェックがない場合、トリガ信号を受信できず、コード印字が行われません。Probe Filterを設定できます。

Level setting：
トリガの極性（Highレベルトリガ／Lowレベルトリガ）を選択します。

Mark times：
Startをクリックした後、センサーが信号をトリガするたびに、システムが設定回数分だけ印字を行います。

TSBOT：
設定した時間内に、センサーが受信したトリガ信号をシステムが自動的に無視（マスク）します。

Start delay：
Startを押してからレーザー出力が始まるまでの遅延時間（ミリ秒）を設定します。

図3-3

## 2.2. フライトマーキング設定

パイプラインの方向を、現場での印字条件に応じて「Left to right（左→右）」または「Right to left（右→左）」から選択します（図3-4）。
左→右　／　右→左
図3-4

Optimize marking order：
「優先順序あり（左）」と「優先順序なし（右）」を比較した例は以下のとおりです。

Enable encoder：
チェックするとエンコーダを使用してパイプライン速度と同期します。チェックしない場合は、シミュレーションパイプライン速度を使用します。

Encoder mode：
Single channel（単相）、Dual channel（2相）、Bidirectional（双方向）の中から選択します（機種によりSingle channelを使用）。

Encoder Settings：

* Measurement length：
  エンコーダに接続されているホイールの周長です。Bearing modeを確認することでパルス数を取得できます。

* Number of pulses：
  エンコーダのパルス数で、エンコーダ本体のラベルから確認できます。

* Move direction：
  「Calculate」と「Tachometer」をクリックすると、自動的に進行方向が正しいか（逆転していないか）を判別します。

* Flight coefficient：
  「Calculate」をクリックするとフライトマーキング係数が自動計算されます（この値は計算値であり、実際の現場マーキング結果に応じて微調整します）。
  調整方法：図1または図2のようなマーキング結果になった場合、現場のレーザー設置方向とパイプライン方向に応じて係数を増減させ、図3のような正しい図形になるまで調整します。

Pipeline Angle：
レーザーとコンベアラインのなす角度です。レーザーマシンとコンベアラインの間にわずかな角度があり、物理的な調整だけでは精度が十分でない場合、ソフトウェア上でこの角度を変更して補正することができます。コンベアラインとレーザーの角度イメージを図3-5に示します。

Force online update：
現在の情報が固定内容（変更される可変データを含まない）の場合でもオンライン更新機能を有効にするためには、「Yes」を選択して強制更新を有効にする必要があります。

Flight compensation mode：
補正方法として「Coordinate position（座標位置）」と「Edge of the lens（レンズ端基準）」から選択します。

Edge distance：
マーキング開始位置を設定します。必要に応じて調整できます。

Turn off over speed alarm：
マーキング速度が速すぎる場合でも、アラームを発生させないようにします。

Fault alarm shutdown：
機械に故障が発生した場合、自動的にシャットダウンします。シャットダウンのタイプを設定できます。

### マーキングモード設定

マーキングモードは、Normal mode、Pipeline mode、Continuous mode の3種類があります。

### 2.2.1. Normal mode

モードをNormal modeに選択し、センサーを有効にします（図3-9）。

Sensor distance：
プローブ距離、つまりプローブ取付位置からマーキング位置までの距離です。

TSBOT：
設定した距離内では、センサーが受信したトリガ信号をシステムが自動的に無視（マスク）します。

図3-9

### 2.2.2. Pipeline mode

マーキングモードでPipeline modeを選択します（図3-10）。

Starting position：
Startボタンをクリックしてから、コンベアがこの距離を移動した時点でコード印字を開始します。

Mark Distance：
直前のマーキング位置から、次のマーキング位置までの距離です。

図3-10

### 2.2.3. Continuous mode

マーキングモードでContinuous modeを選択します（図3-11）。このモードではセンサーを開いておく必要があり、センサー信号は常にHighレベルである必要があります。

Sensor distance：
プローブ距離、つまりプローブ取付位置からマーキング位置までの距離です。

Mark times：
連続印字回数を設定します。制限なしに設定した場合、センサーがHighレベルの間、システムは設定した距離に従って常に印字を行います。

TSBOT：
設定した距離内では、センサーが受信したトリガ信号をシステムが自動的に無視（マスク）します。

Mark Space：
直前のマーキング位置から次のマーキング位置までの間隔距離です。

図3-11
