# 画面構成

ホーム画面は次の要素によって構成されています。

<img src="images/_home_screen_name.jpg" width=400px />


ステータスバー
: マーキングシステムの動作状態やエラー情報などを表示します。刻印操作もここから行います。

メニューバー
: ファイルの作成や保存、操作のやり直し等のツールが配置されています。

オブジェクトパネル
: 加工データの追加を行うことができます。

コントロールパネル
: 加工データの編集操作を行うことができます。

グラフィックビュー
: 加工データがプレビュー表示されています。

#### 1.3.3.11. 塗りつぶし（Fill）

<img src="./images/control_panel/fill.jpg"  width="300px"/>

テキストオブジェクトやベクター図形に塗りつぶしをを設定できます。
単線フォントや開いたパス（始点と終点が繋がっていないパス）には設定できません。

| 項目 | 説明 |
|:---:|-----|
| 塗潰し有効（塗潰し許可） | 有効にすると図形またはテキストの塗りつぶし有効にします。 |

| 輪郭 | 輪郭線の加工有無を設定します。 |
| 塗潰し種類 | 通常（最適化ワイヤー）：一方向に直線を引き重ねて塗りつぶしを行います。輪郭が揃いやすくなります。<br>最適化（ノーマル重点ライン）：双方向に走査して塗つぶしを行います。マーキング時間を短縮できます。<br>弓形（弓形重点ライン）：双方向に走査して塗つぶしを行います。折り返し部分もレーサー照射が行われます。 |

| シングルオフセット | 配列の個々の要素の位置を微調整できます。対象の要素を選択して数値を入力するか、矢印ボタンで移動させます。 |
| 赤ランプ表示 | 選択中の要素をガイド光でプレビュー表示します。 |
