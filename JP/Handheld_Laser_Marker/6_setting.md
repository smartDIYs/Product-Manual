# 3. 設定

## 3.1. マーキングパラメータ

ホーム画面でマーキングファイルを選択し、Setting → Marking parameters をクリックすると、そのマーキングデータのパラメータを変更できます。デフォルト値の変更も可能です。

Advanced：詳細なマーキングパラメータを設定します。
図4-1

Dot times：
単点エネルギーです。フォントがドットマトリクスフォントの場合や、「点」オブジェクトおよびドット配列QRコードを使用する場合に有効になります。出力方式として「点時間出力」または「電気パルス出力」を選択できます。

Marking speed（mm/s）：
マーキング速度。レーザービームの焦点が被マーキング物の表面上を「書いていく」速度を示します。

Jump speed（mm/s）：
ジャンプ速度。ベクター図形の彫刻が終わった後の移動速度を示し、一般的にはマーキング速度の2倍程度に設定します。

Power（%）：
レーザーの相対出力（単位：％）です。値が大きいほど出力が大きくなります。使用時には90％を超えないことを推奨します。

Frequency（KHz）：
レーザー周波数。単位時間あたりのパルス数、すなわち1秒あたりの発光ドット数を示します（単位：KHz）。

Time-lapse parameters（遅延パラメータ）：
遅延パラメータには主に、Jump delay（ジャンプ遅延）、Laser on delay（レーザーオン遅延）、Laser off delay（レーザーオフ遅延）、Marking delay（マーキング遅延）、Corner delay（コーナー遅延）が含まれます。これらは設定されたジャンプ速度およびマーキング速度に合わせて調整する必要があります。
遅延設定が最適化されていない場合、マーキング品質が低下し、マーキング時間も増加します。

一般的に、Laser on delay と Laser off delay の長さは、総スキャン時間には大きな影響を与えません。まずは Laser on delay と Laser off delay を最適化し、その後に Jump delay、Mark end delay、Turning point delay などの「マーク制御系の遅延」を調整することを推奨します。特に Jump delay および Mark end delay を大きめに設定することは有効な場合が多いです。

以下に、各種マーキング遅延がマーキング品質に与える影響を示します。

**Jump delay が短すぎる場合**
Jump delay が短すぎると、ジャンプ後にスキャナヘッドの位置決めが完了する前に最初のマーキングベクターが開始され、図4-2のような振動（ぶれ）現象が発生します。
図4-2

**Jump delay が長すぎる場合**
Jump delay が長すぎると、マーキング品質に顕著な変化はありませんが、マーキング時間だけが長くなります。

**Laser on delay（開光遅延）が短すぎる場合**
Laser on delay が短すぎると、ベクターの開始時点でレンズがまだ所定の角速度に達していないにもかかわらずレーザーが点灯するため、各ベクターの始点が焦げたような状態になります（図4-3）。
図4-3

**Laser on delay（開光遅延）が長すぎる場合**
Laser on delay が長すぎると、ベクターの開始時にレーザーが遅れて点灯するため、ベクターの始点がマーキングされません（図4-4）。
図4-4

**Laser off delay（消光遅延）が短すぎる場合**
Laser off delay が短すぎると、レンズがベクター終端の位置に到達する前に、直線または折れ線の最終マーキングコマンドの後でレーザーが消灯してしまいます。その結果、各ベクターが最後までマーキングされず、図4-5のようになります。
図4-5

**Laser off delay（消光遅延）が長すぎる場合**
Laser off delay が長すぎると、直線または折れ線の最終マーキングが完了した後もレーザーが長く点灯し続け、レンズが停止または非常に低速で移動しているにもかかわらず照射が続きます。その結果、各ベクターの終端部分が焦げたようになります（図4-6）。
図4-6

**Marking delay（マーキング遅延）**
値を大きくしても顕著な品質変化はありませんが、値が大きいほどマーキング時間は長くなります。

**Corner delay（コーナー遅延）が短すぎる場合**
Corner delay が短すぎると、次の折れ線のマーキングコマンドが既に実行されているのに、スキャン範囲がまだ前のベクタの終端位置に達していない状態となり、コーナー部分にアール（丸み）が生じます（図4-7）。
図4-7

**Corner delay（コーナー遅延）が長すぎる場合**
Corner delay が長すぎると、後続のマーキングコマンド中にレンズの動きが非常に遅くなるか、ほとんど停止してしまいます。ベクター間でレーザーがオフにならないため、図4-8のようにコーナー部分が焦げたような状態になります。
図4-8


## 3.2. エリア

### 3.2.1. 振動ミラーのキャリブレーション

Area size：
現在のフィールドレンズにおけるマーキング範囲を示します。

Edit region：
エリアの表示範囲を設定できます。

Mirror 1 = X, Lens 2 = X：
Lens 1 = X を選択し、Manual trigger（手動トリガ）をクリックします。
マーキングされた「ABC」の文字が横向き（水平）であれば選択は正しく、縦向き（垂直）になっていれば選択が誤りです。その場合は Mirror 2 = X を選択し直します。

Correction：
実際のフィールド状況に応じて、マーキングされた「ABC」が反転していないか、回転していないかを確認します。必要に応じて、反転用のミラー設定を選択し、マーキングされた内容が意図どおりになるまで調整します。

係数補正方法：
係数補正の手順は、下表（マニュアル内の表）に示されています。

まず「Debug parameter（デバッグパラメータ）」で「Automatic creation（自動生成）」を選択し、次に「Test parameters（テストパラメータ）」をクリックして矩形サイズを設定します。
例：Area size が 110 の場合、矩形サイズを 109 に設定し、その後「Manual trigger（手動トリガ）」をクリックします。

### 3.2.3. 赤色ガイド光のキャリブレーション

Guide light をクリックすると、図4-9のような赤色ガイド光キャリブレーション画面に入ります。
図4-9

Frame：
「Yes」を選択すると、長方形フレームでガイドします。選択しない場合は、文字アウトライン全体のパスに沿ってガイドします。

Guide speed：
赤色ガイド光の走査線の速度を設定します。

Flight Mode：
赤色ガイド光を縦一本のラインとして表示します。この縦ラインの位置がフライトマーキングの開始位置となるため、開始距離の値を設定する際に便利です。

Red light parameter：
赤色ガイド光に関するパラメータを設定します（下図参照）。

赤色ガイド光のキャリブレーション方法は次のとおりです：

1. 矩形を1つ追加し、それを実際にマーキングします。
2. 赤色ガイド光（Red light guide）をクリックし、オフセットや拡大縮小を調整して、赤色ガイド光の表示と実際にマーキングされた図形とが完全に一致するように合わせます。

赤色ガイド光のフォーカス調整方法は次のとおりです：

1. 赤色フォーカスには2つの赤色光源が必要です。1つは固定、もう1つは調整可能な赤色光とします。
2. 「Red light focus adjustment」にチェックを入れ、上下左右の4つのボタンを調整して、2つの赤色光が完全に重なるように合わせます。

**IO 出力機能の説明：**

* **Serial number end output**：シリアル番号が終了したときに信号を出力します。
* **Fault output**：システムに警告またはエラーが発生したときに信号を出力します。

**Manual output control**：
設定すると、画面上部のバーに有効化ボタンが表示され、ユーザーが手動で出力／無効化を制御できます。

**Pre processing sensor detection**：
マーキングが開始されていない状態でセンサーがトリガされた場合、信号を出力します。

**CTDa**：システム内部用信号。

**Machine working state**：
加工前は導通状態、マーキング処理中は非導通状態になります。

**Power saving management**：
ハンドヘルド機器の省電力制御を行います。

**Output status 0, 1, 2, 3, 4**：
各出力を手動で個別に制御できます。

**例：マーキング完了出力が必要な場合の設定方法**

1. OUT-1 を「Marking completed output（マーキング完了出力）」に設定します。
2. （注：IO出力は Low レベル出力）

   * 接続対象がソリッドステートリレーの場合、リレーの＋端子をインターフェースボードの 5V 出力端子（PL19）に接続し、－端子を 0_1 ポートに接続します。
3. 「Marking completed」の出力パルス幅（信号幅）を設定します。
4. [Save] をクリックし、システムを再起動します。

**IO 入力機能の説明：**

* **Reset serial number**：入力スイッチ信号によりシリアル番号をリセットします。
* **Start/Stop Processing**：入力スイッチ信号で印字の開始／停止を制御します。

### 3.9.1. アップデート

アップデートでは、システムのアップグレード、起動画面（ブートインターフェース）の変更、および本体画面左上に表示される画像の変更を行うことができます。

システムアップグレードの手順は次のとおりです（最初に管理者としてログインしてください）。

1. アップデートファイルをUSBメモリに保存し、そのUSBメモリを本体に挿入します。
2. 「System update」をクリックすると、図4-33のようなポップアップ画面が表示されます。
3. 「Select」をクリックすると、図4-34のようなポップアップ画面が表示されます。
4. 「USB」をクリックしてアップデートファイルを選択し、［OK］をクリックしてから［Start Update］をクリックすると、アップデートが完了します。

起動画面（ブートインターフェース）を変更する手順は次のとおりです。

1. 起動画面用の画像を作成します（形式：bmp、解像度：1280×800、ビット深度：32bit）。
2. 作成した画像のファイル名を「logo」に変更し、USBメモリに保存して本体に挿入します。
3. 「Update logo」をクリックすると、図4-35のようなポップアップ画面が表示されます。
4. 「Select」をクリックし、「USB」をクリックしてアップデートファイルを選択し、［OK］をクリックしてから［Start Update］をクリックすると、起動画面の更新が完了します。

画面左上のロゴを更新する手順は次のとおりです。

1. ロゴ画像を作成します（任意のロゴ画像、画像サイズは25×16mm）。
2. 作成した画像をUSBメモリに保存し、本体に挿入します。
3. 「M-LOGO」をクリックすると、図4-35と同様のポップアップ画面が表示されます。
4. 「Select」をクリックし、「USB」をクリックしてアップデートファイルを選択し、［OK］をクリックしてから［Start Update］をクリックすると、画面左上のロゴの更新が完了します。

### 3.9.2. Registered

ソフトウェアの使用時間または使用日数を制限するための機能です。

手順は次のとおりです。

**Step 1：マシンコードをUSBメモリにエクスポートする**

1. 本体の電源を入れ、システムにログインします（パスワード：123）。
2. 「Setting → System Info → Export Machine Code」をクリックし、画面の指示に従って操作を完了します。

**Step 2：登録ソフトを起動する**
登録ソフトウェアを起動します（登録ソフトはレーザーメーカーに別途配布されています）。

**Step 3：マシンコードを入力する**
マシンコードを入力します。
（注：マシンコードは操作画面からUSBメモリへ直接エクスポートできます。画面表示の指示に従って操作を完了してください。）

**Step 4：キーを設定する**
キー（Key）を設定します。これはそのマシン専用の鍵となるため、必ず控えておいてください。次回以降の登録時にも同じキーを使用する必要があります。

**Step 5：試用期間または永久使用を設定する**
ソフトウェアの試用期間、または永久使用かを設定します。

**Step 6：登録コードを生成してUSBメモリに保存する**
登録コードを生成し、USBメモリに保存します。

**Step 7：本体側で登録を行う**

1. 管理者ユーザーでログインします（パスワード：123）。
2. 「Settings → System Info」をクリックし、USBメモリにTXTファイルとして保存しておいた登録コードをインポートします。
3. 「Registered」をクリックして登録を完了します。
